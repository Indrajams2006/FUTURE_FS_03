# Nira Collection Website

## Live Demo
https://nira-shine-studio.lovable.app

## About
A landing page built for Nira Collection, a real jewelry business specializing in anti-tarnish, lightweight, daily-wear jewelry. Built using Lovable (AI website builder).

## Tools Used
Lovable (AI Website Builder)

## Pitch
Nira Collection had an Instagram presence but no dedicated website, meaning potential customers browsing beyond Instagram had no easy way to discover their full catalog or brand story. I built a clean, professional landing page showcasing their 6 product categories (Necklaces, Earrings, Bracelets, Rings, Anklets, Gift Sets), brand highlights (anti-tarnish, lightweight, long-lasting), and a gallery of their pieces — all directing customers straight to their Instagram DM to order. This gives Nira Collection a credible, shareable web presence that builds trust with new customers before they even message, while keeping their existing Instagram-based ordering process intact.
